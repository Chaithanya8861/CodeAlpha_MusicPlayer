# 🎵 Melodify - Modern Music Player

A feature-rich, web-based music player application with a beautiful glass-morphism interface, playlist management, audio controls, and autoplay functionality. Built with pure HTML, CSS, and JavaScript.

## ✨ Features

### Core Audio Controls
- **Play/Pause** – Start and pause the current track
- **Next/Previous** – Navigate through your playlist
- **Progress Bar** – Click anywhere on the progress bar to seek
- **Volume Control** – Adjust volume with slider or mute/unmute
- **Real-time Display** – Shows current time and total duration

### 🎯 Advanced Features
- **📋 Dynamic Playlist** – View all songs with their artists and durations
- **🔁 Autoplay Mode** – Toggle on/off for continuous playback
- **🎨 Now Playing Display** – Shows album art, song title, and artist
- **⌨️ Keyboard Support** – Control the player without touching the mouse
- **💾 Smart State Management** – Remembers playing state across track changes

### 🎨 UI/UX Highlights
- **Glass-morphism Design** – Modern translucent effects with backdrop blur
- **Smooth Animations** – Hover effects, transitions, and visual feedback
- **Responsive Layout** – Works perfectly on desktop, tablet, and mobile
- **Visual Progress Tracking** – Real-time progress bar updates

## 🛠️ Technologies Used

- **HTML5** – Semantic structure and audio element integration
- **CSS3** – Modern styling with Flexbox, Grid, gradients, and animations
- **Vanilla JavaScript** – Audio control logic, playlist management, and event handling

## 🚀 How to Use

### Mouse/Touch Controls
1. **Play a song** – Click the play button (▶) or click on any song in the playlist
2. **Change volume** – Drag the volume slider or click the speaker icon to mute
3. **Seek within song** – Click anywhere on the progress bar
4. **Skip tracks** – Use the next (⏭) and previous (⏮) buttons
5. **Toggle autoplay** – Click the autoplay toggle switch (🔁)

### Keyboard Shortcuts
| Key | Action |
|-----|--------|
| `Spacebar` | Play/Pause |
| `Arrow Left` | Previous track |
| `Arrow Right` | Next track |
| `Arrow Up` | Increase volume |
| `Arrow Down` | Decrease volume |
| `M` | Mute/Unmute |

### Autoplay Behavior
- **When ON** – Automatically plays the next song when current track ends (loops through playlist)
- **When OFF** – Stops playback at the end of the current song
